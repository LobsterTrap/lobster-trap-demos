# Lobster Trap Demos

Hands-on demos and tutorials for running [OpenClaw](https://github.com/openclaw/openclaw) in containers.

## Demos

| Demo | Description |
|------|-------------|
| [Local to K8s and Back](local-to-k8s-and-back/) | Containerize an OpenClaw agent, deploy to Kubernetes, and come back to local — without changing your config. |

## Contributing

Have a demo idea? Open an issue or submit a PR with a new folder. Each demo should include a README, reproducible scripts, and step-by-step instructions.
